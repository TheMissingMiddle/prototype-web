# prototype-web
web prototype (not using ionic)

## Objectives

 - Maximum legacy support w/ Vue + Socket.io (IE 8/+, Android 2.2/+, Most other browsers)
 - Ease of use
 - Complex things get done in client-side logic
 - Minimal jQuery usage
